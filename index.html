Анимации исчезновения

Вставка в Genially (Iframe)

Код сжат с помощью LZ-алгоритма. Ссылка стала короче в 3 раза специально для Genially! Вставьте через Insert -> Other -> Iframe.

Альтернатива: Отдельный файл

Если Genially всё ещё жалуется на длину (из-за огромного количества вопросов), просто скачайте игру отдельным файлом, закиньте её к себе на GitHub и вставляйте прямую короткую ссылку на него!
